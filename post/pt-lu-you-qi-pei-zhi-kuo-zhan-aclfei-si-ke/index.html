PT路由器配置扩展ACL（非思科）

2022-05-01


在路由器R1配置模式下输入命令
access-list 102 permit tcp host 192.168.40.100 host 192.168.30.100 eq ftp
access-list 102 deny tcp host 192.168.40.101 host 192.168.50.100 eq www
access-list 102 permit ip any any
